Thesis Statement

  • Designing experiences for virtual reality (VR), like any other type of design, involves the fundamental aspect of standardization, which makes interfaces more familiar and thus usable.

  • Just like how the dropdown is a universal web element that is widely used to present the user with an array of options, there are very few established design patterns for VR.

  • This is especially true for narrative experiences. Core functions like selecting scenes in a menu, pausing and playing, and selecting choices for feedback are either varied, causing disconnected experiences, or non-existent.

  • My project will explore these core interactions, which many of these experiences share, in order to investigate how best to accommodate the userâ€™s needs when using the application.
